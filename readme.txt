【日時管理プラグイン dtmng ver1.0.0】

====================================================================================================
■ 概要
====================================================================================================
面倒な日時の管理を簡単に扱えるようにするプラグインです。
ゲーム内の日付や実際の日時を管理することができます。
また、時間を加算したり、経過時間や差分を計算することも可能です。

====================================================================================================

  data/others/plugin/dtmng/dtmng.js
2.first.ksからプラグインを呼び出してください。
  [plugin name=dtmng]
3.後述のタグリファレンスを参考にして日時管理を行います。

■ タグリファレンス（ティラノスクリプト用）※ティラノビルダー用のコンポーネントリファレンスは後述

; 下記リファレンスの1～7の内容を、ティラノスクリプトのシナリオファイル（.ks）に、そのまま貼り付けても動作します。

;----------------------------------------------------------------------------------------------------
; 1.日時設定
;
; 日時の初期値を設定し、日時管理に日時を追加します。
;
; 【タグ】
; dtmng_set
;
; 【パラメータ】
; id    省略可能 日時を識別する一意キー（省略時はnow）
; scope 省略可能 日時を格納する変数のスコープ「sf,f,tfを指定可能」（省略時はf）
; d     省略可能 日付（省略時は現在日付）
; t     省略可能 時刻（省略時は現在時刻）
;----------------------------------------------------------------------------------------------------
①「dtmng_set」タグを使って、日時の初期値を設定します。[r][r]
【1.パラメータなしで設定】[r]
[dtmng_set]
パラメータを省略したので、システム日付の「[emb exp="f.dtmng.datetime"]」が設定されました。[l][r]

[r]

【2.ID指定で設定】[r]
[dtmng_set id="start"]
ID="start"に「[emb exp="f.dtmng.datetime"]」が設定されました。[l][r]

[r]

【3.日時指定で設定】[r]
[dtmng_set id="start" d="20300401" t="000000"]
ID="start"に日時を指定したので「[emb exp="f.dtmng.datetime"]」が設定されました。[l][r]

[dtmng_set id="end" d="20310228" t="131530"]
ID="end"に日時を指定したので「[emb exp="f.dtmng.datetime"]」が設定されました。[r]

[p]

;----------------------------------------------------------------------------------------------------
; 2.日時取得
;
; 日時管理から日時を取得します。
;
; 【タグ】
; dtmng_get
;
; 【パラメータ】
; id    省略可能 日時を識別する一意キー（省略時はnow）
; scope 省略可能 日時を格納する変数のスコープ「sf,f,tfを指定可能」（省略時はf）
;----------------------------------------------------------------------------------------------------
②「dtmng_get」タグを使って、日時を取得します。[r][r]
[dtmng_get]
ID未設定(now)を取得したので、日時は「[emb exp="f.dtmng.datetime"]」です。[l][r]

[dtmng_get id="start"]
ID="start"を取得したので、日時は「[emb exp="f.dtmng.datetime"]」です。[l][r]

[dtmng_get id="end"]
ID="end"を取得したので、日時は「[emb exp="f.dtmng.datetime"]」です。[r]

[p]

取得した日時を利用する項目は、指定した変数スコープのdtmngオブジェクトに格納されます。[r]

[r]

(sf or f or tf).dtmng.idでIDを表示：[emb exp="f.dtmng.id"][r]

(sf or f or tf).dtmng.datetimeで日時を表示：[emb exp="f.dtmng.datetime"][r]
(sf or f or tf).dtmng.jdatetimeで日時を表示：[emb exp="f.dtmng.jdatetime"][r]
(sf or f or tf).dtmng.datetime_num_stringで日時を表示：[emb exp="f.dtmng.datetime_num_string"][r]

(sf or f or tf).dtmng.dateで日付を表示：[emb exp="f.dtmng.date"][r]
(sf or f or tf).dtmng.jdateで日付を表示：[emb exp="f.dtmng.jdate"][r]
(sf or f or tf).dtmng.date_num_stringで日付を表示：[emb exp="f.dtmng.date_num_string"][r]

(sf or f or tf).dtmng.timeで時刻を表示：[emb exp="f.dtmng.time"][r]
(sf or f or tf).dtmng.jtimeで時刻を表示：[emb exp="f.dtmng.jtime"][r]
(sf or f or tf).dtmng.time_num_stringで時刻を表示：[emb exp="f.dtmng.time_num_string"][r]

[p]

(sf or f or tf).dtmng.yで年を表示：[emb exp="f.dtmng.y"][r]
(sf or f or tf).dtmng.jyで年を表示：[emb exp="f.dtmng.jy"][r]

(sf or f or tf).dtmng.mで月を表示：[emb exp="f.dtmng.m"][r]
(sf or f or tf).dtmng.jmで月を表示：[emb exp="f.dtmng.jm"][r]

(sf or f or tf).dtmng.dで日を表示：[emb exp="f.dtmng.d"][r]
(sf or f or tf).dtmng.jdで日を表示：[emb exp="f.dtmng.jd"][r]

(sf or f or tf).dtmng.hrで時を表示：[emb exp="f.dtmng.hr"][r]
(sf or f or tf).dtmng.jhrで時を表示：[emb exp="f.dtmng.jhr"][r]

(sf or f or tf).dtmng.minで分を表示：[emb exp="f.dtmng.min"][r]
(sf or f or tf).dtmng.jminで分を表示：[emb exp="f.dtmng.jmin"][r]

(sf or f or tf).dtmng.secで秒を表示：[emb exp="f.dtmng.sec"][r]
(sf or f or tf).dtmng.jsecで秒を表示：[emb exp="f.dtmng.jsec"][r]

[p]

(sf or f or tf).dtmng.wで曜日を表示：[emb exp="f.dtmng.w"][r]
(sf or f or tf).dtmng.jwで曜日を表示：[emb exp="f.dtmng.jw"][r]
(sf or f or tf).dtmng.w_numで曜日を表示：[emb exp="f.dtmng.w_num"][r]
※w_numは0：日曜 ～ 6：土曜の数値になります。[r]

[p]

;----------------------------------------------------------------------------------------------------
; 3.日時加算
;
; 日時に年月日時分秒を加算（減算）します。
;
; 【タグ】
; dtmng_add
;
; 【パラメータ】
; id      省略可能 日時を識別する一意キー（省略時はnow）
; scope   省略可能 日時を格納する変数のスコープ「sf,f,tfを指定可能」（省略時はf）
; years   省略可能 加算年（省略時は0）
; months  省略可能 加算月（省略時は0）
; days    省略可能 加算日（省略時は0）
; hours   省略可能 加算時（省略時は0）
; minutes 省略可能 加算分（省略時は0）
; seconds 省略可能 加算秒（省略時は0）
;----------------------------------------------------------------------------------------------------
③「dtmng_add」タグを使って、日時を加算します。[r][r]
[dtmng_get id="start"]
ID="start"の日時は「[emb exp="f.dtmng.datetime"]」です。[l][r]
[dtmng_add id="start" days=3]
ID="start"に3日足したので、日時は「[emb exp="f.dtmng.datetime"]」です。[l][r]
[dtmng_add id="start" months=-5]
ID="start"から5ヶ月引いたので、日時は「[emb exp="f.dtmng.datetime"]」です。[r]

[p]

;----------------------------------------------------------------------------------------------------
; 4.日時経過
;
; id1とid2の日時の経過を取得します。
;
; 【タグ】
; dtmng_past
;
; 【パラメータ】
; id1    省略可能 日時を識別する一意キー（省略時はnow）
; scope1 省略可能 日時を格納する変数のスコープ「sf,f,tfを指定可能」（省略時はf）
; id2    省略可能 日時を識別する一意キー（省略時はnow）
; scope2 省略可能 日時を格納する変数のスコープ「sf,f,tfを指定可能」（省略時はf）
;----------------------------------------------------------------------------------------------------
④「dtmng_past」タグを使って、日時の経過を取得します。[r][r]
[dtmng_get id="start"]
ID="start"の日時は「[emb exp="f.dtmng.datetime"]」です。[l][r]
[dtmng_get id="end"]
ID="end"の日時は「[emb exp="f.dtmng.datetime"]」です。[l][r]
[dtmng_past id1="start" id2="end"]
経過日時は「[emb exp="tf.jpastDateTime"]」です。[r]

[p]

経過日時は必ず、一時変数（tf）に格納されます。[r]

[r]

経過日時は「[emb exp="tf.jpastDateTime"]」です。[r]

[r]

tf.pastYearsで経過年の部分を表示：[emb exp="tf.pastYears"][r]
tf.jpastYearsで経過年の部分を表示：[emb exp="tf.jpastYears"][r]
tf.pastMonthsで経過月の部分を表示：[emb exp="tf.pastMonths"][r]
tf.jpastMonthsで経過月の部分を表示：[emb exp="tf.jpastMonths"][r]
tf.pastDaysで経過日の部分を表示：[emb exp="tf.pastDays"][r]
tf.jpastDaysで経過日の部分を表示：[emb exp="tf.jpastDays"][r]

[p]

経過日時は「[emb exp="tf.jpastDateTime"]」です。[r]

[r]

tf.pastHoursで経過時間の部分を表示：[emb exp="tf.pastHours"][r]
tf.jpastHoursで経過時間の部分を表示：[emb exp="tf.jpastHours"][r]
tf.pastMinutesで経過分の部分を表示：[emb exp="tf.pastMinutes"][r]
tf.jpastMinutesで経過分の部分を表示：[emb exp="tf.jpastMinutes"][r]
tf.pastSecondsで経過秒の部分を表示：[emb exp="tf.pastSeconds"][r]
tf.jpastSecondsで経過秒の部分を表示：[emb exp="tf.jpastSeconds"][r]

[p]

経過日時は「[emb exp="tf.jpastDateTime"]」です。[r]

[r]

tf.pastDateTimeで経過日時を表示：[emb exp="tf.pastDateTime"][r]
tf.jpastDateTimeで経過日時を表示：[emb exp="tf.jpastDateTime"][r]
tf.pastDateで経過日付の部分を表示：[emb exp="tf.pastDate"][r]
tf.jpastDateで経過日付の部分を表示：[emb exp="tf.jpastDate"][r]
tf.pastTimeで経過時刻の部分を表示：[emb exp="tf.pastTime"][r]
tf.jpastTimeで経過時刻の部分を表示：[emb exp="tf.jpastTime"][r]

[p]

■ 使い方（ティラノビルダー用）
====================================================================================================
※ティラノビルダー用は「dtmng.tbp」ファイルを使用します。
1.「dtmng{バージョン}.zip」を展開してください。
2.ティラノビルダーのメニューから、「プロジェクト」-「プラグイン」を選択しプラグイン画面を表示してください。
3.プラグイン画面で「プラグインを追加」ボタンを押して展開したフォルダ内の「dtmng.tbp」を選択してください。
4.プラグインがインストールされると、新規プラグイン「日時管理プラグイン」インストール完了と表示されます。
5.「プラグインを追加」ボタンの下にあるリストに「日時管理プラグイン」が追加されます。
6.プラグイン画面を閉じます。
7.ティラノビルダーのメニューから、「プロジェクト」-「コンポーネント管理」を選択しコンポーネント管理画面を表示してください。
8.コンポーネント管理画面を下へスクロールし、「プラグイン」をチェックしてください。
9.「日時設定」「日時取得」「日時加算」「日時経過」「日時差分」「日時削除」「日時クリア」の7つのコンポーネントがチェックされていることを確認してください。
10.コンポーネント管理画面の下部にある「適用する」ボタンを押してください。
11.ティラノビルダーのメイン画面に戻り、コンポーネントの最下部にプラグインが表示されているか確認してください。
12.後述のコンポーネントリファレンスを参考にして日時管理を行います。

■ コンポーネントリファレンス（ティラノビルダー用）
====================================================================================================
;----------------------------------------------------------------------------------------------------
; 1.日時設定
;
; 日時の初期値を設定し、日時管理に日時を追加します。
;
; 【コンポーネント】
; 日時設定
;
; 【パラメータ】
; 変数格納スコープ  日時を格納する変数のスコープ「sf,f,tfを指定可能」（初期値はf）
; ID              日時を識別する一意キー（初期値はnow）
; 日付            日付（初期値は空白）
; 時刻            時刻（初期値は空白）
;----------------------------------------------------------------------------------------------------
「日時設定」コンポーネントをシナリオエリアにドラッグ＆ドロップし、各パラメータを設定します。
※この例では３つ用意しています。

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  【１つ目】現在のシステム日時
  変数格納スコープ：ゲーム変数(f)
  ID：now
  日付（yyyyMMdd 数値８桁）：空白
  時刻（hhmmss 数値６桁）：空白
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~  
  【２つ目】2030年4月1日 0時0分0秒
  変数格納スコープ：ゲーム変数(f)
  ID：start
  日付（yyyyMMdd 数値８桁）：20300401
  時刻（hhmmss 数値６桁）：000000
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  【３つ目】2031年2月28日 13時15分30秒
  変数格納スコープ：ゲーム変数(f)
  ID：end
  日付（yyyyMMdd 数値８桁）：20310228
  時刻（hhmmss 数値６桁）：131530
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

;----------------------------------------------------------------------------------------------------
; 2.日時取得
;
; 日時管理から日時を取得します。
;
; 【コンポーネント】
; 日時取得
;
; 【パラメータ】
; 変数格納スコープ  日時を格納する変数のスコープ「sf,f,tfを指定可能」（初期値はf）
; ID              日時を識別する一意キー（初期値は0）
;----------------------------------------------------------------------------------------------------
「日時取得」コンポーネントをシナリオエリアにドラッグ＆ドロップし、各パラメータを設定します。
※今回は「日時設定」で設定した３つの日時を取得して表示します。

  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  【１つ目】現在のシステム日時
  ・「日時取得」コンポーネント
  　変数格納スコープ：ゲーム変数(f)
  　ID：now
  ・「テキスト」コンポーネント
  　１つ目の日時は[emb exp="f.dtmng.datetime"]です。
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~  
  【２つ目】2030年4月1日 0時0分0秒
  ・「日時取得」コンポーネント
  　変数格納スコープ：ゲーム変数(f)
  　ID：start
  ・「テキスト」コンポーネント
  　２つ目の日時は[emb exp="f.dtmng.datetime"]です。
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
  【３つ目】2031年2月28日 13時15分30秒
  ・「日時取得」コンポーネント
  　変数格納スコープ：ゲーム変数(f)
  　ID：end
  ・「テキスト」コンポーネント
  　３つ目の日時は[emb exp="f.dtmng.datetime"]です。
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

;----------------------------------------------------------------------------------------------------
; 3.日時加算
;
; 日時に年月日時分秒を加算（減算）します。
;
; 【コンポーネント】
; 日時加算
;
; 【パラメータ】
; 変数格納スコープ  日時を格納する変数のスコープ「sf,f,tfを指定可能」（初期値はf）
; ID              日時を識別する一意キー（初期値は0）
; 加算年          加算する年（省略時は0）
; 加算月          加算する月（省略時は0）
; 加算日          加算する日（省略時は0）
